# 一级指针

```c
int *p;
p = &a;

// *p = a 值
// p = &a 空间
```



# 二级指针(指向指针的指针)

```c
int a = 10;
int *p = &a;
int **pp = &p;

// 那么
pp = &p = &&a;
*pp = p = &a;
**pp = *p = a
```

## 二级指针与数组

```c
char *a[5] = {"aaa", "bbb", "ccc"};
char **p;
p = a;

// 则默认 *p = p[0] = a[0] = "aaa";
```







+++

*p++ 的问题

`*` 与`++`是属于同一级的运算符，故从左到右运算 *p++ = *p, p++

`*p+1` = (*p) + 1 ：`*`运算级别比 `+` 高
