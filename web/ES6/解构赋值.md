# 数组解构赋值

1. 基础用法

   ```javascript
   const [a, b, c] = [1, 2, 3];
   ```

2. 默认值

   ```javascript
   const [a = 1, b = 2, c = 3] = [10];
   
   console.log(a, b, c); // 10 2 3
   ```

3. 变换变量值

   解构赋值可以方便地交换两个变量的值，而**无需使用临时变量**。

   ```javascript
   let x = 1;
   let y = 2;
   [x, y] = [y, x];
   console.log(x); // 2
   console.log(y); // 1
   ```

4. 嵌套数组解构

   ```javascript
   const [a, [b, c]] = [1, [2, 3]];
   console.log(a, b, c); // 1 2 3
   ```

5. 剩余元素

   使用剩余元素语法...可以将数组中的剩余元素收集到一个新的**数组**中。

   ```javascript
   const [a, ...rest] = [1, 2, 3, 4];
   console.log(a); // 1
   console.log(rest); // [2, 3, 4]
   ```



# 对象的解构赋值

1. 基本用法

   ```javascript
   const { a, b } = { a: 1, b: 2 };
   console.log(a, b); // 1 2
   ```

2. 默认值

   ```javascript
   const { a = 10, b = 5 } = { a: 3 };
   console.log(a, b); // 3 5
   ```

3. 重命名变量

   ```javascript
   const { a: x, b: y } = { a: 1, b: 2 };
   console.log(x); // 1
   console.log(y); // 2
   ```

4. 嵌套对象解构

   ```javascript
   const {
     a,
     b: { c, d }
   } = {
     a: 1,
     b: { c: 2, d: 3 }
   };
   console.log(a, c, d); // 1 2 3
   ```

5. 剩余属性

   使用剩余属性语法`...`可以将对象中的剩余属性收集到一个新的**对象**中。

   ```javascript
   const { a, ...rest } = { a: 1, b: 2, c: 3 };
   console.log(a); // 1
   console.log(rest); // { b: 2, c: 3 }
   ```
