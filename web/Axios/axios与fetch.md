# fetch和xhr

| 特性       | **Fetch**                        | **XMLHttpRequest (XHR)** |
| ---------- | -------------------------------- | ------------------------ |
| 类型       | 现代 API，**基于 Promise**       | 传统 API，**基于回调**   |
| 简洁性     | 更简洁，符合现代 JavaScript 风格 | API 较复杂，代码更冗长   |
| 浏览器支持 | 现代浏览器，IE 不支持            | IE5 及以上支持           |

#### **流式处理**

- Fetch

  支持 `ReadableStream`，**适合处理大文件或流式响应**

- XHR

  只能等待完整响应返回后再处理，**不能流式处理数据**
