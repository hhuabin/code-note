# 分支

查看状态：**切换分支前，一定要查看状态，确认所有内容已提交再切换**

```shell
git status
```

新建分支

```shell
git branch branchName
```

查看分支

```shell
git branch -v
```

切换分支

```shell
git checkout branchName
```

合并分支

```shell
git merge branchName
```

修改分支名字

```shell
git branch -M newBranchName
```

删除分支

```shell
git branch -d branchName
```

强行删除分支

```shell
git branch -D branchName
```
